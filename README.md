# Boids
Predator -Prey Simulation using Thoth and PrimeMover

## Build Status
![Build Status](https://github.com/hellblazer/Boids/actions/workflows/maven.yml/badge.svg)
___

This is an example of an event driven simulation of a predator-prey interaction using Thoth as the area of interest managment and PrimeMover as the simulation framework.

Licensed under the AGPL V3

Build with Maven 3.x:

	mvn clean install
